Na disciplina de programação web, desenvolvemos uma API Rest voltada à gestão de castração de animais e outros tipos de manejo por meio de agendamentos. O intuito do projeto era que ele servisse de apoio às atividades de uma ONG da cidade de Avanhandava. 
O projeto foi desenvolvido na linguagem Java com uso dos frameworks Spring e Hibernate. O sistema foi estruturado sob a arquitetura Model-View-Controller (MVC). Os testes são realizados por meio do Postman, já que as operações de CRUD da API são através de métodos HTTP.
